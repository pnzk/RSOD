# RSOD
Official implementation of **RSOD**, a diagnosis-and-mitigation framework for context overfitting in remote-sensing few-shot object detection.

This repository includes:

- **Paired out-of-context diagnostic protocol**
- **Context-preserving RSOD augmentation**
- **RVLP-LCC prototype calibration**
- **RSF-Adapter feature adaptation**
- **Training and evaluation configurations**
- **Reproducibility materials for DIOR and NWPU VHR-10 experiments**
